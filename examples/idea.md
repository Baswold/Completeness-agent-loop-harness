# Example Project: Simple Calculator

## Goal
Create a simple command-line calculator application in Python.

## Requirements

### Core Features
1. Support basic arithmetic operations: add, subtract, multiply, divide
2. Accept input from command line arguments
3. Handle division by zero gracefully
4. Support floating point numbers

### Code Quality
1. Include proper error handling
2. Write unit tests for all operations
3. Add docstrings to all functions
4. Follow PEP 8 style guidelines

### Output Format
- Display the operation and result clearly
- Show error messages for invalid input

## Example Usage
```bash
python calculator.py add 5 3
# Output: 5 + 3 = 8

python calculator.py divide 10 0
# Output: Error: Cannot divide by zero
```

## Deliverables
- calculator.py - Main application
- test_calculator.py - Unit tests
- README.md - Usage documentation
